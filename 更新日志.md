# 更新日志

所有的重要更改都将记录在此文件中。

## [1.0.1] - 2026-03-15

### ✨ 新增功能
*   **管理后台控制面板 (Dashboard)**：新增可视化统计大屏，实时展示项目数量、留言总数、待审核留言及今日新增数据。
*   **中文化应用日志**：自定义 Gin 日志格式，所有请求访问信息、错误提示及启动日志均已实现全中文输出。
*   **首页交互增强**：实现首页锚点平滑滚动效果，通过 JS 拦截导航，提升交互质感。

### 🛠️ 路由与体验优化
*   **URL 地址精简**：通过后端路由重写，移除了所有 `.html` 后缀及冗余路径，实现 `/`/`/admin`/`/messages` 等简洁跳转。
*   **全局资源路径扁平化**：配置 `/globals` 静态托管，简化前端对通用 JS 和 CSS 的引用路径。
*   **SEO 深度补全**：为所有前端页面添加了 SEO 元数据（Title, Description, Keywords），并在后台管理页面禁用了搜索引擎抓取以保证私密性。

### 🐛 修复与重构
*   **URL 清洁化**：解决了首页通过 JS 进行 `location.href` 跳转导致的 URL 冗余问题。
*   **代码清理**：修复了后端 main.go 中的 import 冲突及 lint 错误，确保编译成功。
*   **导航修复**：统一了所有页面的 `<a>` 标签跳转逻辑，确保在无后缀情况下依然能正确渲染页面。

## [1.0.0] - 2026-03-15

### ✨ 核心功能
*   **前后端分离架构**：基于 Go (Gin) 后端 API 与原生 HTML/CSS/JS 前端的轻量级系统实现。
*   **作品集展示**：支持动态加载个人项目列表及图文详情页。
*   **互动留言板**：内置数学题验证码及频率限制保护的留言系统。
*   **全功能后台**：提供完整的管理员登录、留言审核/回复、作品管理及图片上传功能。
*   **安全机制**：集成 JWT 身份验证、Bcrypt 密码加密及接口级请求频率保护。
