## [1.1.0] - 2026-03-21

### 🚀 架构与部署大升级 (SQLite + Docker)
- **数据库架构切换**：全面告别 MySQL，转为使用 **SQLite**。现在核心数据存放在 `storage/jeje.db` 文件中，彻底消除了外部数据库依赖。
- **环境配置极简化**：`.env` 文件由复杂的数据库连接配置精简为仅需一个文件名，实现“解压即运行”。
- **容器化一键部署**：
    - **Docker 支持**：新增 `Dockerfile` 和 `docker-compose.yml`。
    - **持久化方案**：通过挂载 `/app/storage` 目录，同时实现数据库文件与上传媒体的双重持久化。
    - **编译优化**：后端构建新增 CGO 支持，完美驱动 SQLite 高性能读写。
- **查询语法适配**：重构了 `FIND_IN_SET` 等 MySQL 特有语法，确保在非 MySQL 环境下的搜索兼容性。

## [1.0.9] - 2026-03-21

### 🖼️ 作品媒体展示升级
- **响应式图片轮播 (Carousel)**：引入了全新的作品展示组件，支持同一个项目展示多张高清图片。支持自动播放（6s 间隔）、左右切换及进度原点跳转，且完美适配移动端。
- **视频功能大爆发**：
    - **格式支持扩展**：新增对 `.mov`, `.avi`, `.mkv`, `.m4v` 等多种三方/高清视频格式的直接上传与播放识别。
    - **1GB 超大上传**：将单次文件上传限制从 50MB 极大提升至 1GB，彻底解决高清演示视频上传失败的问题（需配合重启后端生效）。
    - **播放体验优化**：新增 `playsinline` 属性解决移动端强制全屏问题，并在视频加载失败时自动提供源码下载/新窗口打开链接。

### 🎨 管理体验与资源优化
- **全站 100% 本地化 (脱离 CDN)**：移除了所有外部 Font Awesome 链接，改用基于 CSS Data URI 的 **SVG 本地图标引擎**，彻底告别国内访问 CDN 慢导致的编辑器图标白块问题。
- **媒体管理闭环**：在作品编辑页新增了封面图与视频的“设为首页预览”、“删除”及“一键复制地址”快捷操作。
- **时间维度精确化**：全站所有日期展示（包括详情页与留言板）现已精确到 **时、分、秒**。

### ⚙️ 后端逻辑补全
- **URL 智能校验**：通过重构 `isSafeURL` 逻辑，现在支持在封面图/视频字段中填入多行或逗号分隔的多个合法链接。
- **配置一致性**：同步了后端配置文件（.env）、系统默认配置与前端 UI 展示的上传大小限制。
