# JEJE WEB - 个人网站

JEJE WEB 是一个采用精致浅灰色调、设计现代且功能完备的个人网站系统。项目采用前后端分离架构，并深度优化了路由与交互体验，支持全中文日志输出，是一个极致纯粹且视觉高端的个人展示空间。

## 🌟 项目亮点

*   **精致视觉体系**：全站采用经过微调的浅灰色调（#f8f9fa），配合分层设计提升页面深度，视觉体验更加高级。
*   **极简路由**：全面移除 `.html` 后缀，支持 `/admin`、`/messages` 等现代简洁 URL 地址。
*   **上传进度可视化**：后台视频上传支持实时百分比进度条反馈，确重大文件操作的确定感。
*   **数据大屏**：内置管理员控制面板，实时统计项目总数、留言频率及待审项，全局掌控运行状态。
*   **全站 100% 本地化 (No CDN)**：移除了所有外部脚本、字体与图标链接。采用了基于 CSS Data URI 的 **SVG 本地图标引擎**，彻底告别国内访问 CDN 的不稳定性，实现毫秒级加载。
*   **作品图片轮播 (Carousel)**：内置响应式详情页轮播系统，支持同一个项目展示多张图片、自动切换与进度跳转，完美提升作品展示力。
*   **超强视频支持**：
    *   **1GB 超大容量**：上调了全站上传上限（1GB），支持高清作品演示。
    *   **多格式兼容**：支持 `.mp4`, `.mov`, `.avi`, `.mkv` 等多种格式。
    *   **智能兜底**：配备了播放进度条可视化及播放失败自动转下载链接的逻辑。
*   **项目标签管理**：后台可维护标签，前台支持标签筛选。
*   **全中文日志与时间**：全站时间展示精确到 **时:分:秒**。后端日志全中文化，便于监控和维护。
*   **富交互体验**：
    *   **平滑滚动**：首页导航支持无感丝滑滚动，且地址栏状态保持极致纯净。
    *   **实时进度条**：演示视频上传过程实时百分比显示，上传完成后自动平滑隐藏。
    *   **响应式 UI**：完美适配 PC 与移动端。
*   **SEO 深度优化**：每个页面均配置完善的元数据标签，助力搜索引擎收录。

## 🛠️ 技术栈

*   **数据库**：SQLite (文件驱动，零配置)
*   **容器化**：Docker, Docker Compose
*   **后端**：Go (Gin framework), SQLx, JWT, Bcrypt
*   **前端**：原生 JavaScript (ES6+), Vanilla CSS, Google Fonts

## 📂 简洁目录

```text
jeje_web/
├── backend/            # 后端集成代码 (SQLite 驱动)
├── frontend/           # 前端静态资源 (SPA 风格简洁 URL)
├── storage/            # 运行时存储 (数据库 jeje.db 与上传媒体)
├── Dockerfile          # 容器构建蓝图
└── docker-compose.yml  # 一键部署配置
```

## 🚀 部署指南 (Docker 推荐)

如果您服务器已有 Docker 环境，这是最快捷的部署方案：

1.  **准备环境**：将项目文件夹上传至服务器。
2.  **配置密钥**：修改 `docker/docker-compose.yml` 中的 `JWT_SECRET` 为一个随机字符串。
3.  **一键启动**：
    在项目根目录下执行：
    ```bash
    docker-compose -f docker/docker-compose.yml up -d --build
    ```
4.  **访问**：打开 `http://YOUR_SERVER_IP:8080` 即可。

## 🛠️ 本地开发

1.  在 `backend` 目录下执行 `go mod tidy`。
2.  直接运行 `npm run dev` 或 `go run main.go`。
3.  系统会自动在 `storage/` 目录下生成 `jeje.db` 数据库文件。
